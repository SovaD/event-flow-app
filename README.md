## EventFlow — платформа для организации событий

EventFlow — это современное React-приложение, предназначенное для упрощения процесса планирования мероприятий. 


## Основные возможности

На основе концепции проекта, описанной в документации:

* Современный UI/UX дизайн, вдохновленный принципами чистоты и комфорта.
* Единая база гостей, которую нужно заполнить лишь однажды.
* Интуитивно понятное создание карточек событий с описанием и таймингом.
* Генерация персональных ссылок для подтверждения присутствия гостями.
* Защита данных пользователей с использованием JWT-авторизации.

## Технологический стек

* Frontend: React.js
* Иконки: Lucide-React
* Стилизация: Встроенный CSS (Scoped styles) для простоты развертывания.
* Дизайн-система: Glassmorphism, CSS Gradients.


## Инструкция организатора

* Авторизация: Войдите в систему (можно использовать демо-данные из раздела "О платформе").
* Контакты: Настройте свою базу VIP-гостей.
* Событие: Создайте новый ивент и прикрепите к нему гостей.
* Приглашение: Скопируйте RSVP-ссылку и отправьте её гостю.
